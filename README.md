# de3_api_docker

Data Engineering 3 API project packaged with Docker.

## Project Goal

This repository contains a containerized API assignment for Data Engineering 3. The focus is on packaging an API so it can be built and run consistently with Docker.

## Files

- `de3_artyom_ashigov_assignment.pdf` - assignment/report document.

## Running

Build and start the Dockerized project:

```bash
docker compose up --build
```

If the repository is run without Docker Compose, use the command specified in the assignment PDF or application source.

## What This Demonstrates

- API packaging
- Docker-based reproducibility
- Basic deployment workflow for a data engineering service

## Tools

Docker, Docker Compose, and API development tools.
